# React tic-tac-toe app

![Screenshot of Tic Tac Toe App](screenshot.png)

## To Run Locally

Clone the project

```bash
  git clone https://github.com/onetrickollie/React_TicTacToe.git

Go to the project directory

```bash
  cd .../TicTacToe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
