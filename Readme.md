
# React tic-tac-toe app

![Screenshot of React Tic Tac Toe App](screenshot.png)

## To Run Locally

Clone the project

```bash
  git clone https://github.com/jahnltib/react-tic-tac-toe
```

Go to the project directory

```bash
  cd .../react-tic-tac-toe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
